--------------------------------- Proyecto Integrador ---------------------------------------

-- ¿En qué consiste el Proyecto Integrador? -------------------------------------------------

El desarrollo del presente proyecto integrador tiene como fin procesar la data (movie_dataset.csv) 
la cual se halla en el repositorio de github "rashida048". Todo esto mediante el uso de los
conocimientos adquiridos en la materia de base de datos, aplicando proceso de elaboración de tablas 
(entidades), de procesos de optimización de dichas entidades y de la llevada al aprendizaje y uso 
de mysql para la programación de dichas entidades, atributos y carga de data.
